# For My Love ❤️ - A National Love Day Gift

This is a special, verified website made for your girlfriend.

## How to Run
1. Open this folder in your terminal.
2. Run `npm install` (if you haven't already).
3. Run `npm run dev`.
4. Open the link shown (usually `http://localhost:5173/`).

## How to Customize
You can edit the text in the following files to make it even more personal:
- **Hero/Title**: `src/components/Hero.jsx`
- **The Letter**: `src/components/Letter.jsx`
- **Poetry**: `src/components/Poetry.jsx`
- **Prayers**: `src/components/Prayers.jsx`
- **"Proud Of You" Paragraph**: `src/components/ProudOfYou.jsx`

## Features
- **Responsive Design**: Looks good on mobile and desktop.
- **Glassmorphism**: Modern, premium aesthetic.
- **Animations**: Elements float and fade in beautifully.
- **Hearts**: Floating hearts in the background.
